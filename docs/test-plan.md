# Test Plan

## 1. Объект тестирования
Веб-приложение с функционалом регистрации и авторизации

## 2. Цель тестирования
Проверка корректности основных пользовательских сценариев

## 3. Область тестирования
- Регистрация пользователя
- Авторизация пользователя

## 4. Типы тестирования
- Функциональное тестирование
- Негативное тестирование

## 5. Критерии начала тестирования
- Приложение доступно
- Отображаются формы регистрации и логина

## 6. Критерии завершения тестирования
- Все тест-кейсы выполнены
- Критические ошибки отсутствуют

## 7. Риски
- Ошибки валидации данных
- Ошибки при логине
